# Steve's Word Manuscript Template

This is the R Markdown template I use for journals that insist on
Microsoft Word submissions. The functionality is limited, mostly because
Microsoft Word is limited. The only real use here is the effective
anonymity that emerges in the Word document when the primary workflow is
still built around compilation to a LaTeX PDF.

## Usage

``` r
word(...)

templ_word()
```

## Arguments

- ...:

  Arguments to
  [`rmarkdown::word_document`](https://pkgs.rstudio.com/rmarkdown/reference/word_document.html).
