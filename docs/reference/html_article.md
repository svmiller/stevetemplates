# Steve's HTML Article Template

This is the R Markdown template I added in support of another package,
which facilitates the conversion of academic articles to HTML format.

## Usage

``` r
html_article(...)

templ_html_article()
```

## Arguments

- ...:

  Arguments to
  [`rmarkdown::word_document`](https://pkgs.rstudio.com/rmarkdown/reference/word_document.html).
